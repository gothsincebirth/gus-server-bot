# requirement
Discord.py for Linux and MacOS: 
```
python3 -m pip install -U discord.py
``` 

Discord.py for Windows:
``` 
py -3 -m pip install -U discord.py
``` 

Python-dotenv download
```
pip install python-dotenv
```
